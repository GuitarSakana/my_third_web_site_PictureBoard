# my_third_web_site_PictureBoard

--------------- 습득한 지식 및 구현 --------------------

* html canvas태그 공부 및 활용
* javaScript로 상호작용 하며 canvas 기능 구현 (주로 javascript로 작동한다.)     ex) 각종 이벤트 리스너, 위치 좌표 가져와서 캔버스에 그리기.
* javaScript로 그리기, 채우기, 지우기, 사진 불러오기, 사진 저장하기 기능 구현     ex) url사용해서 이미지 불러오기, 저장하기.
* javaScript로 선, 도형, 텍스트를 canvas에서 사용할 수 있게 하였고, 굵기와 사이즈를 개별적으로 적용하여 사용할 수 있게 구현.
* css와 javaScript의 상호 작용으로 내가 무엇을 선택하여 사용하고 있는지 시각적으로 알 수 있게 구현.(선택표시)


---------------아쉬운점, 부족한점 ----------------------

* 사진 저장하기에서 url을 사용하는 부분에 대한 이해가 부족.
* 부모 자식간에 태그 관계에 대한 유연한 사고 부족.  ex) 태그 안에서 자체적으로 a태그를 사용해서 동작 등..
* css에 대해서 더 많은 지식과 정확한 이해 필요. (반응형 디자인)



---------------------- 차후 계획 -------------------------

* 백엔드(spring)를 공부해서 디벨럽 할 예정.
* spring 프레임워크의 소켓통신을 주로해서 유명한 게임인 Catch Mind를 만들어 볼 예정.
* Catch Mind프로젝트를 진행하면서 그에 맞게 다시 css도 바꿀 예정.
